Enforcing a single instance for Java app
-----------------------------------------

A zero-external-dependency and lightweight library for enforcing a single instance on a Java app. 

It includes the support of custom URI-scheme handler, which is useful for opening your app using your 
own custom URI scheme like `backdoor://auth?key=1234`. A custom URI-scheme is commonly used for authenticating 
users through your website's authentication system.
